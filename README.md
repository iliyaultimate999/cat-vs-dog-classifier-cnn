# Cat vs Dog Binary Classifier
A fundamental Computer Vision project utilizing Convolutional Neural Networks (CNN).

### Project Overview
This project serves as a baseline for binary image classification, focusing on model architecture design and pixel data processing.

### Key Features
- Custom CNN architecture including Convolutional, Pooling, and Dropout layers.
- Binary Cross-Entropy optimization for binary classification tasks.
- End-to-end image preprocessing and normalization pipeline.

### Tech Stack
- TensorFlow / Keras
- Matplotlib
